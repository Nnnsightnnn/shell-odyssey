# System Technical Notes

## Power Systems

The Odyssey uses a dual-core fusion reactor with redundant power distribution.

- Primary reactor output: 847 MW nominal
- Secondary backup: 200 MW emergency
- EPS grid voltage: 4.7 kV standard

**Note:** Always verify containment field integrity before adjusting reactor output.

## Life Support

Atmospheric processors maintain Earth-normal conditions:

- O2: 21% ± 0.5%
- N2: 78% ± 1%
- Pressure: 101.3 kPa
- Temperature: 22°C ± 2°C

**Note:** Backup systems can sustain crew for 72 hours in emergency mode.

## Navigation

Current position calculations require:

1. Star chart reference (updated weekly)
2. Subspace beacon triangulation
3. Inertial navigation backup

**Note:** Manual course corrections should be verified by two officers.

## Communications

Subspace communication range depends on local conditions:

- Clear space: ~50 light-years
- Near stellar objects: ~5 light-years
- Emergency beacon: Always active

**Note:** Message compression reduces bandwidth by 80% for routine traffic.
